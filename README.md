# TypeRenew-plugins

TypeRenew 官方插件仓库 —— 为 TypeRenew 提供官方专用拓展集合

截止 2026/4/15，项目自带四个 TypeRenew 专用拓展，安装后即可启用：

| 插件 | 当前版本 | 功能 |
|------|------|------|
| RenewAvatar | 1.1.0 | 头像源替换，支持多种 Gravatar 镜像源，可全局生效或仅限评论区域 |
| RenewGo | 1.1.0 | 外链安全拓展，支持安全提示页或 302 跳转，可配置白名单和日志记录 |
| VditorRenew | 1.1.0 | 引入 Vditor 编辑器，支持所见即所得、即时渲染、分屏预览三种模式，兼容旧文章 |
| RenewSEO | 1.1.0 | 集成 Robots、Sitemap、Canonical、OG、时间因子、主动推送与 SEO 监控等能力 |
| RenewShield | 1.1.0 | 提供基础 WAF、限频拦截、搜索引擎放行、访问控制、上传防护、健康检查与运行日志等能力。**注意：该插件仅用于基础防护，帮助新手站长过滤常见机器人和恶意请求；如站点已部署专业防护软件，或已接入 CDN、云 WAF诶中 等服务，通常无需安装。** |
